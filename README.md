## 🎯 Project Overview
An end-to-end clinical intelligence system that transforms raw medical data into actionable survival insights. This project focuses on identifying key clinical factors associated with mortality in heart failure patients by combining **Python-driven data engineering** with **Advanced Power BI visualization**.

---

## ✨ Key Highlights & Findings
* **Survival Rate:** 67.9% overall survival rate among the study group.
* **Critical Risk Driver:** Patients with high **Serum Creatinine** levels show a **+49% higher mortality rate**.
* **Hypertension Impact:** 63% of recorded deaths were associated with hypertensive patients.
* **Data Integrity:** 100% automated data cleaning pipeline developed in Python.

---

## 🏗️ Technical Architecture & Workflow

### 1. Data Engineering (Python)
* **Cleaning:** Handled missing values and standardized categorical variables (Sex, Smoking, Diabetes).
* **EDA:** Performed statistical analysis using `Pandas`, `Matplotlib`, and `Seaborn` to find correlations between biomarkers.
* **Feature Scaling:** Prepared clinical indicators (Ejection Fraction, Platelets) for visual representation.

### 2. Business Intelligence (Power BI)
* **DAX Modeling:** Created complex measures for Survival Rate and Mortality KPIs.
* **Interactive UI:** Developed a glassmorphism-inspired dashboard for filtering by (Age, Sex, Clinical Condition).
* **Clinical Profiling:** Visualized the impact of Chronic diseases (Anaemia, Diabetes) on patient outcomes.

---

## 🛠️ Tech Stack
* **Core Language:** Python (Pandas, NumPy, Seaborn ,matplotlib)
* **Environment:** Google Colab
* **BI Tool:** Power BI Desktop
* **Data Source:** Heart Failure Clinical Records (CSV)
